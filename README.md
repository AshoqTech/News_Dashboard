# 📰 News Dashboard

A modern and responsive **News Dashboard web application** built with React, JavaScript, and Tailwind CSS. It fetches and displays real-time news data using APIs and presents it in a clean, user-friendly dashboard interface.

## 🚀 Live Demo

🌐 https://news-dashboard-liart-omega.vercel.app/

---

## 📖 Overview

This project is a fully functional News Dashboard designed to provide users with up-to-date news in an organized layout. It demonstrates real-world API integration, dynamic rendering, and responsive UI design.

The application runs locally with API configuration and is deployed on Vercel for live preview.

---

## ✨ Features

- 📰 Real-time news fetching via API
- 📱 Fully responsive design (mobile + desktop)
- 🎨 Clean and modern UI using Tailwind CSS
- ⚡ Fast performance with React
- 🔄 Dynamic content rendering
- 🗂️ Organized news layout/cards
- 🌐 Deployed on Vercel

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/AshoqTech/newsdashboard.git
```

### Navigate to project

```bash
cd newsdashboard
```

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```
